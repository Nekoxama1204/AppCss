# Actividad 2.4 - Aplicaciones con CSS

Este proyecto presenta la integración de dos animaciones profesionales creadas con CSS, HTML y JavaScript, encapsuladas en una interfaz moderna y responsiva utilizando Bootstrap 5.

## Elementos Incluidos
1. **Magic Social Share Menu**: Menú de redes sociales desplegable de forma circular con animaciones de transición en `hover`.
2. **Animated Navbar**: Barra de navegación animada con un indicador dinámico, respuesta al pasar el cursor y un bloqueo temporal de 2 segundos al hacer clic.

## Tecnologías Utilizadas
* HTML5
* CSS3
* JavaScript (Vainilla)
* Bootstrap 5.3.2
* Ionicons 7.1.0
